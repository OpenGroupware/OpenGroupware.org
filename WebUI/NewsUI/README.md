# NewsUI - News/Announcements UI

NewsUI provides the web interface for the OGo newsboard
where users can read and post announcements.

**Bundle:** `NewsUI.lso` (5 source files)


## Dependencies

- OGoFoundation
- LSFoundation (LSNews commands)


## Key Components

| Component | Purpose |
|-------------------------------|---------------------------|
| `SkyNews`                     | News article listing      |
| `LSWNewsArticleViewer`        | View article              |
| `LSWNewsArticleEditor`        | Create/edit article       |
| `SkyNewsPreferences`          | News preferences          |
