# JobUI - Task Management UI

JobUI provides the web interface for managing tasks
(jobs), including listing, editing, status tracking,
and project assignment.

**Bundle:** `JobUI.lso` (12 source files)


## Dependencies

- OGoFoundation
- LSFoundation (LSTasks commands)


## Key Components

| Component | Purpose |
|----------------------------------|---------------------------|
| `LSWJobs`                        | Task listing page         |
| `LSWJobViewer`                   | View task details         |
| `LSWJobEditor`                   | Edit task                 |
| `LSWJobAction`                   | Task status actions       |
| `LSWJobPreferences`              | Task preferences          |
| `SkyJobList`                     | Modern task list          |
| `OGoJobStatus`                   | Status display            |
| `SkyJobAttributesEditor`         | Edit task attributes      |
| `SkyAssignProjectToJobEditor`    | Link task to project      |
| `LSWJobMailPage`                 | Email task link           |
