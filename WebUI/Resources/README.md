# Resources - Localization Strings

Resources contains localization/translation files
(`.strings` files) for the OGo web interface.


## Supported Languages

| Directory | Language |
|---------------------------|---------------------------|
| `English.lproj`           | English (default)         |
| `German.lproj`            | German                    |
| `French.lproj`            | French                    |
| `Spanish.lproj`           | Spanish                   |
| `Italian.lproj`           | Italian                   |
| `Dutch.lproj`             | Dutch                     |
| `Danish.lproj`            | Danish                    |
| `Portuguese.lproj`        | Portuguese                |
| `ptBR.lproj`              | Brazilian Portuguese      |
| `Swedish.lproj`           | Swedish                   |
| `Polish.lproj`            | Polish                    |
| `Hungarian.lproj`         | Hungarian                 |
| `Japanese.lproj`          | Japanese                  |
| `Basque.lproj`            | Basque                    |
| `Finnish.lproj`           | Finnish                   |
| `Norwegian.lproj`         | Norwegian                 |
| `Slovak.lproj`            | Slovak                    |

Each directory contains `.strings` files with
key-value pairs for UI labels, messages, and other
localizable text.
