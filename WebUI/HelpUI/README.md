# HelpUI - Integrated Help System

HelpUI provides the built-in help system for the OGo
web interface.

**Bundle:** `HelpUI.lso` (6 source files)


## Dependencies

- OGoFoundation


## Key Components

| Component | Purpose |
|---------------------------|-------------------------------|
| `OGoHelpDatabase`         | Help content management       |
| `OGoHelpSection`          | Help topic sections           |
| `OGoHelpPage`             | Individual help pages         |
| `OGoHelpViewer`           | Help display component        |
| `OGoHelpAction`           | Help direct actions           |

Configuration: `product.plist`
