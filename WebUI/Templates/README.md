# Templates - HTML Templates

Templates contains SOPE/WebObjects template files
(`.wox` format) for all WebUI bundles. Templates are
organized in subdirectories matching their bundle names.

Template files use the SOPE `.wox` XML-based template
format with WebObjects-style bindings for dynamic
content generation.


## Subdirectories

Each WebUI bundle has a corresponding template
directory:

- `AddressUI/`, `PersonsUI/`, `EnterprisesUI/`
- `BaseUI/`, `OGoUIElements/`
- `PreferencesUI/`, `AdminUI/`, `GroupsUI/`
- `LSWScheduler/`, `OGoScheduler/`,
  `OGoSchedulerViews/`, `OGoResourceScheduler/`,
  `OGoSchedulerDock/`
- `LSWMail/`, `OGoWebMail/`, `OGoMailEditor/`,
  `OGoMailFilter/`, `OGoMailInfo/`,
  `OGoMailViewers/`, `OGoRecipientLists/`
- `OGoNote/`, `LSWProject/`, `OGoProject/`,
  `OGoProjectInfo/`, `OGoProjectZip/`,
  `OGoDocInlineViewers/`
- `HelpUI/`, `JobUI/`, `NewsUI/`, `RegUI/`
- `LDAPAccounts/`, `OpenGroupware/`, `Themes/`
