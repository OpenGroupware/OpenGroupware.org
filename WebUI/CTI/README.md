# CTI - Computer-Telephony Integration

CTI provides telephony integration for the OGo web
interface, enabling click-to-dial functionality.


## Sub-Bundles

### AsteriskDialer

Asterisk VoIP PBX dialer integration. Allows users
to initiate phone calls through an Asterisk server
directly from the OGo contact interface.

### STLIDialer

STLI (Standard Telephony Line Interface) dialer
integration for traditional telephony systems.
